# PolyglotNote-Automator-
一个高效的语言学习辅助工具。通过全局快捷键捕获选中文本，调用 Google 翻译获取多词性详细释义，并自动排版生成美观的 Word 学习笔记，支持唤起本地词典。 A high-efficiency language learning assistant that captures selected text, fetches multi-POS definitions via Google Translate API, and auto-formats professional MS Word study notes with a single hotkey.
